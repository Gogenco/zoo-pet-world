# Stage 4.7D — Manual Object Hit Zones

## Что было не так

Stage 4.7C с pixel detection засчитывал клики по реальной разнице пикселей.
Это работало, но диапазон был слишком широкий:
- можно было попасть по кусту вместо цветка;
- дерево/фон давали слишком много невнятных отличий;
- игроку было непонятно, куда именно нажимать.

## Что исправлено

Pixel detection для PNG-уровней отключён.
Добавлены ручные object hit zones для всех 16 уровней.

Теперь игра принимает клики только по понятным объектам:
- облако;
- цветок;
- мяч;
- бабочка;
- дерево;
- чашка;
- жучок;
- рыбка;
- ракушка;
- ягоды;
- листья;
- и т.д.

## Главный фикс для Level 1 Lion

Для льва зоны теперь:
1. бабочка;
2. облако;
3. дерево/крона;
4. цветы слева;
5. мяч.

Цветок засчитывается по зоне цветка, а не по всему кусту.
Облако имеет более крупную честную зону.
Дерево имеет понятную большую зону на кроне/стволе.

## Что НЕ менялось

- PNG визуалы не менялись;
- 16 уровней сохранены;
- rewards/unlock не менялись;
- hint/restart/zoom/pan не менялись;
- victory modal не трогался;
- другие игры не трогались.

## Проверки

- Inline JS syntax: OK
- levels: 16
- diffs: 5 на каждом уровне
- manual hitZones: есть у всех уровней
