# STAGE 4.6E — Find Differences Visual Rework

## Что сделано
В мини-игре **«Найди отличия»** полностью переработаны **визуалы сцен** для 4 уровней, строго по ТЗ:

- **животные не менялись** — используются текущие игровые PNG-ассеты;
- переделаны **только сцены, окружение, декоративные объекты, фон и композиция**;
- сохранены:
  - механика поиска отличий,
  - координаты попаданий,
  - hint/restart,
  - zoom/pan,
  - блокировка/открытие уровней,
  - награды,
  - общий flow мини-игры.

## Что обновлено по сценам
### 1. Lion / Саванна
- новый мягкий sky gradient;
- более аккуратные холмы и слои земли;
- декоративное дерево с веткой;
- солнце с friendly-стилем;
- кусты, камни, цветок, мяч, бабочка.

### 2. Panda / Бамбуковый сад
- новый светлый фон и layered ground;
- бамбуковые стебли и листья;
- чашка, камень, насекомое, декоративный лист;
- сцена стала чище и насыщеннее.

### 3. Dolphin / Море
- многослойное море и берег;
- лодка, пузырьки, ракушка, рыба, морская звезда;
- улучшен casual-mobile вид сцены.

### 4. Parrot / Джунгли
- новый джунглевый фон;
- лиана, крупный лист, цветок, ягоды, дальние деревья;
- сцена стала более сочной и детской.

## Отличия
Во всех уровнях сохранено по **5 отличий**. Отличия сделаны через изменение:
- цвета объектов;
- появления / исчезновения объекта;
- смещения объекта;
- уменьшения / увеличения объекта.

## Что НЕ менялось
- логика игры;
- структура прогресса;
- completion flow;
- другие режимы.
