# Stage 4.4C — Coloring canvas size and non-paintable background fix

## База
`ZooPetWorld_stage4_4B_coloring_and_pairs_map_fix.zip`

## Что исправлено

### 1. Размер картинки в раскраске
- внутренний canvas переведён в портретный формат;
- сцена теперь занимает почти всю рабочую область по высоте и ширине;
- изображение рисуется крупнее и ближе к краям окна;
- старый большой цветной пустой фон вокруг картинки убран.

### 2. Внешние поля больше не закрашиваются
- если вокруг картинки остаются небольшие поля, они теперь блокируются в boundary canvas;
- ребёнок может закрашивать только саму картинку, а не внешний фон вокруг неё.
