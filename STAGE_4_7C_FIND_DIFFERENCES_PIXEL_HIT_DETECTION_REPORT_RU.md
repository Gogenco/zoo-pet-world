# Stage 4.7C — Find Differences Pixel Hit Detection Fix

## Что было не так

В 4.7A/4.7B координаты отличий пытались рассчитываться заранее.
На premium PNG-картинках это ненадёжно:
- AI-картинки дают много мелких отличий не там, где игрок их ожидает;
- автоматический анализ иногда выбирал животное/фон вместо настоящего объекта;
- маркер появлялся не на том месте.

## Что исправлено

Для PNG-уровней добавлен новый алгоритм:
- клик проверяется по реальной разнице пикселей между left/right картинками;
- если в месте клика реально есть визуальное отличие — игра засчитывает его;
- маркер ставится именно туда, куда нажал игрок;
- старые автоматические hitZones для PNG-уровней больше не используются.

## Что осталось прежним

- визуалы не менялись;
- 5 отличий на уровень;
- rewards/unlock;
- hint/restart;
- zoom/pan;
- victory modal;
- другие мини-игры.
