# Stage 4.6A — Pet Room Decor MVP

## База

- Базовый архив: `ZooPetWorld_stage4_5F_stabilization_build.zip`
- Цель этапа: добавить маленькую безопасную progression-фичу в комнату питомца без новых мини-игр и без риска для стабилизации.

## Что добавлено

Добавлен MVP интерьера комнаты питомца:

- 3 предмета декора:
  - Цветок — 60 монет
  - Кровать — 120 монет
  - Игрушки — 90 монет
- Предметы видны в комнате, но сначала закрыты замком.
- Тап на закрытый предмет открывает confirm popup покупки.
- Покупка списывает монеты только после подтверждения.
- Если монет не хватает, показывается small coin popup.
- Открытые предметы сохраняются и остаются в комнате после перезагрузки.
- Добавлен маленький статус: `Интерьер комнаты: X/3 предметов открыто`.

## Что специально НЕ добавлено

Чтобы не ломать игру, в этом MVP не добавлены:

- fake rewarded ads;
- скидка за рекламу;
- пассивный доход;
- влияние на hunger/mood;
- уникальные предметы для каждого животного;
- сложные анимации;
- патч поверх `renderPetRoom` через таймеры.

## Архитектура

Добавлены отдельные функции с безопасным неймингом:

- `renderPetRoomDecor()`
- `buyPetRoomDecorItem()`
- `petRoomDecorUnlocked`

Система аккуратно оборачивает текущий `renderPetRoom()` и после стандартного рендера рисует только декор.

## Save keys

Новый ключ:

- `zooPetRoomDecorUnlocked`

Формат:

```json
{
  "plant": true,
  "bed": true,
  "toys": true
}
```

Если ключа нет, все 3 предмета остаются закрытыми, игра не падает.

## Popup / покупка

Добавлен safe fallback для `showPurchaseConfirm`, потому что в текущей базе использовался вызов, но полноценной глобальной функции не было.

Особенности:

- не используется browser `confirm()`;
- overlay имеет класс `purchase-confirm-overlay`;
- карточка имеет класс `purchase-confirm-card`;
- есть крестик `×`;
- есть `Отмена` и `Подтвердить`;
- `Esc` и overlay close работают через уже существующий modal-audit.

Также добавлен fallback `showCoinShortPopup`, если функция отсутствует.

## Что проверить вручную

1. Открыть `Мой питомец`.
2. Увидеть 3 предмета с замками.
3. Тапнуть на закрытый предмет.
4. Проверить confirm popup.
5. Нажать `Отмена` — монеты не списываются.
6. Нажать `Подтвердить` при достаточных монетах — предмет открывается.
7. Проверить, что монеты списались один раз.
8. Перезагрузить страницу — предмет остаётся открытым.
9. При нехватке монет появляется small popup.
10. Проверить, что кормление / игра / сон / душ питомца работают как раньше.
11. Проверить, что Раскраска всё ещё красит.
12. Проверить, что Найди пары и Пазл открываются.
