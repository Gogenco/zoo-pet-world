# Stage 4.7B — Find Differences Hit Zones Fix

## Что исправлено

В Stage 4.7A визуалы были добавлены правильно, но координаты отличий были рассчитаны слишком грубо.
Из-за этого пользователь мог нажимать на реально отличающийся объект, но игра не засчитывала клик.

Stage 4.7B исправляет именно это.

## Изменения

- Для всех 16 PNG-уровней пересчитаны зоны попаданий.
- Добавлены расширенные `hitZones` для каждого отличия.
- `findSpotDiffAtPoint()` теперь проверяет:
  - основную точку отличия;
  - дополнительные hit-зоны вокруг реальных визуальных отличий.
- Радиусы зон сделаны более forgiving, чтобы на мобильном было удобно попадать пальцем.

## Что НЕ трогалось

- PNG-визуалы не менялись.
- Порядок уровней не менялся.
- rewards/unlock не менялись.
- hint/restart/zoom/pan не менялись.
- victory modal не менялся.
- другие мини-игры не трогались.

## Проверки

- External JS syntax: OK
- Inline JS syntax: OK
- levels: 16
- diffs: 5 на каждом уровне
- hitZones: есть у всех отличий
